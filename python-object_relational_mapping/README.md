# Python - Object-relational mapping

## Təsvir
Bu layihədə Python-un `MySQLdb` modulu və `SQLAlchemy` ORM (Object Relational Mapper) vasitəsilə verilənlər bazası ilə işləməyi öyrənirik.

## Öyrənilənlər
* Python ilə MySQL serverinə qoşulma.
* SQL sorğularını Python daxilində icra etmə.
* Cədvəlləri Python klasslarına (ORM) uyğunlaşdırma.
